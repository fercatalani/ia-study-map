### MÊS 1 - FUNDAMENTOS (Arquitetura + Backend sólido)

  * MVC vs Clean Architecture
  * Separação de responsabilidades
* Criar API simples em Node.js (CRUD básico)

* Aprender:

  * Camadas (controller, service, repository)
  * Boas práticas (DTOs, validação)
* Refatorar API aplicando arquitetura limpa

### MÊS 2 - PRÁTICA (pensamento de sênior)

#### SEMANA 5-6:

* Adicionar banco de dados (PostgreSQL + ORM)
* Implementar autenticação (JWT)

#### SEMANA 7-8:

* Trabalhar com:

  * Logs e tratamento de erros
  * Estrutura escalável de pastas
* Integrar frontend (Next.js) com backend

### MÊS 3 - PORTFÓLIO E PREPARAÇÃO

#### SEMANA 9-10:
